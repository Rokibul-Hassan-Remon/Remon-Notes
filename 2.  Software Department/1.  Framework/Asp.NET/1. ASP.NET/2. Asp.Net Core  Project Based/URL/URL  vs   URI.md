### URL ---->  Protocol + URI 
#### URI --->  website address 


##  Example:

**URI :**
		**website address**  :  ` www . javatpoint . com `

**URL**
		**Website Protocol :** `http / https `
,
				 **URI :**   `www.javatpoint.com` 
		
So , ultimately **URL** = **Protocol + URI**  =  ` https//www.javatpoint.com `
